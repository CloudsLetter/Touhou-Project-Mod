### Touhou Project Mod
该项目为 TH06 -> TH19 提供四项基础功能修改，目前仅包含 锁残、锁Bomb、满能量及锁定能量、无敌 等基础功能，当然不仅限于此，后续功能仍在开发中敬请期待。欢迎提交 PR 或 Issues

使用 C# 编写，通过外部对应用内存R/W实现功能。支持中/英/日语言，其中日语完全使用机器翻译，使用Visual Stuido 2022 Preview 及 WPF开发 DONET 版本为8.0
